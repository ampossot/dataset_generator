The Chi-Square (χ²) Test: A Statistical Test for Experiments

Use this module to read the lab instructions, follow the workflow guide, launch the three virtual chi-square tools, and access the assignment for submission.

📘 Lab ManualRead null hypotheses, expected values, χ² calculations, df, p-values, and interpretation.🌱 Dihybrid χ² ToolGenerate AaBb × AaBb data and test 9:3:3:1 expectations.🧪 Test-Cross χ² ToolGenerate AaBb × aabb data and test 1:1:1:1 expectations.🪰 Drosophila χ² ToolGenerate Drosophila population data and test independent assortment.📝 AssignmentOpen Assignment #6.

How students should use this module

1

Read the chi-square introduction

Review H₀, H₁, observed vs. expected values, χ², degrees of freedom, p-values, and conclusions.

2

Analyze TLC results

Collect last week's TLC plate, observe pigment bands under UV light, and use those observations for Assignment #6.

3

Use Tool 1: Dihybrid χ²

Generate a dihybrid dataset, calculate expected values for 9:3:3:1, run χ², and interpret the result.

4

Use Tool 2: Test-Cross χ²

Generate a test-cross dataset, calculate expected values for 1:1:1:1, run χ², and calculate recombination frequency if needed.

5

Use Tool 3: Drosophila χ²

Generate a wild population, perform the trihybrid test cross, choose two traits, and test independent assortment.

6

Submit the assignment

Complete Assignment #6 and submit a physical copy or scanned PDF through Canvas.
